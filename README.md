# Hachure organization profile

This repository owns the public profile shown on the
[`hachure-org`](https://github.com/hachure-org) organization page.

- [`profile/README.md`](profile/README.md) explains Hachure's purpose, links to
  the normative specification and package, and records the current governance
  posture.
- [`hachure-org/spec`](https://github.com/hachure-org/spec) owns the normative
  schemas, derivation and merge semantics, conformance vectors, and bundled
  implementation. This profile repository must not restate those contracts as
  a competing specification.
- [`hachure.org`](https://hachure.org) is the public website and live format
  demonstration.

Keep the profile concise and update it whenever repository names, package
surfaces, implementation links, or governance statements change.
